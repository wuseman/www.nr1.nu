# ncftpget

##### Internet file transfer program for scripts that can be easily used to get files from a ftp

   ncftpget  -u Us3r -p Passw0rd ftp.backups.com . 'site.com.tar.gz'

   ncftpget  -u Us3r -p Passw0rd ftp.backups.com . 'site.com.tar.gz'
