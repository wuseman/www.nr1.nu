# umount

##### umount all nfs mounts on machine

   umount  -a -t nfs

##### Unmount all mounted SAMBA/Windows shares

   umount  -t smbfs

##### Unmount locked filesystems.

   umount  -l /media/foo

##### Unmount all CIFS drives

   umount  -a -t cifs

##### umount all nfs mounts on machine

   umount  -a -t nfs
