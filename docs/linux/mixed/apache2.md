# apache2

##### Apache server config file

   apache2 ctl -V | grep SERVER_CONFIG_FILE

##### Restart web server

   apache2 ctl graceful

##### Know which modules are loaded on an Apache server

   apache2  -t -D DUMP_MODULES

##### See loaded modules in apache(2)

   apache2  -l
