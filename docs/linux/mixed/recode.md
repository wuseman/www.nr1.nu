# recode

##### Modificar o enconding de todos arquivos de um diret?rio

   recode 

##### Show available conversions

   recode  -l |less

##### txt2html

   recode  ..HTML < file.txt > file.html

##### Convert file from UTF8 (no BOM) to UTF16 (with BOM)

   recode  UTF8..UTF-16LE linux-utf8-file.txt
