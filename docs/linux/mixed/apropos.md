# apropos

##### greps the man pages to find utilities

   apropos  keyword

##### search the manual page names and descriptions

   apropos  somekeyword

##### Get full from half remembered commands

   apropos  <part_rember> | less

##### List Network Tools in Linux

   apropos  network |more
