# tidy

##### Adding formatting to an xml document for easier reading

   tidy  -i -xml <inputfile>

##### Prettify an XML file

   tidy  -xml -i -m [file]

##### Convert HTML file into valid XML

   tidy  -asxhtml -numeric < index.html > index.xml
