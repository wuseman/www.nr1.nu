# gpsbabel

##### send a .loc file to a garmin gps over usb

   gpsbabel  -D 0 -i geo -f "/path/to/.loc" -o garmin -F usb:

##### Convert KML to GPX w/ gpsbabel

   gpsbabel  -i kml -f in.kml -o gpx -F out.gpx
