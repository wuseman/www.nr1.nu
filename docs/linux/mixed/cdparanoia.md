# cdparanoia

##### Rip audio tracks from CD to wav files in current dir

   cdparanoia  -B

##### rip all tracks of the cd then convert all *wav files in *mp3

   cdparanoia  -wB 1-; for i in *.wav; do lame -h -b 192 "$i" "`basename "$i" .wav`".mp3; done
