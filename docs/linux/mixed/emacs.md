# emacs

##### Open a file at the specified line

   emacs  +400 code.py

##### open emacs within terminal

   emacs  -nw

##### open emacs with no init file

   emacs  -q

##### emacs: byte compile file

   emacs  batch -Q -no-site-file -eval '(byte-compile-file "my-file.el")'

##### Talk to the doctor (Eliza-like)

   emacs  <ESC+x> doctor
