# bzgrep

##### Fast search in man files or bz-files by keyword direct by man or bz files

   bzgrep  -lE "COMMIT_EDITMSG" /usr/share/man/man?/git*

##### search the pattern from bzip2'ed  file

   bzgrep  -i "pattern" pattern.bz2
