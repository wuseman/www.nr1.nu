# rlwrap

##### Make any command read line enabled (on *nix)

   rlwrap  sqlite3 database.db

##### MoscowML with editable input-line and history

   rlwrap  mosml

##### GNU screen internal console

   rlwrap  -S "$STY> " sh -c 'while read; do screen -S "'"$STY"'" -X $REPLY; done'
